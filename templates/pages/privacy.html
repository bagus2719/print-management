Kebijakan Privasi

Terakhir diperbarui: 28 April 2026

1. Informasi yang Kami Kumpulkan

Saat Anda menggunakan layanan PrintManager, kami mengumpulkan informasi berikut:

  • Informasi Akun: Username dan password yang Anda buat saat registrasi.
  • File Dokumen: File PDF dan gambar yang Anda unggah untuk dicetak.
  • Data Transaksi: Riwayat pesanan, konfigurasi cetak, dan informasi pembayaran.
  • Data Teknis: Alamat IP, jenis browser, dan informasi perangkat untuk keperluan keamanan.

2. Penggunaan Informasi

Informasi yang kami kumpulkan digunakan untuk:

  • Memproses dan menyelesaikan pesanan cetak Anda.
  • Mengelola akun dan autentikasi pengguna.
  • Memproses pembayaran melalui gateway Midtrans.
  • Berkomunikasi dengan Anda terkait status pesanan.
  • Meningkatkan kualitas layanan kami.

3. Pengelolaan File Dokumen

Penting:

  • File yang Anda unggah hanya digunakan untuk keperluan pencetakan.
  • File akan dihapus secara permanen dari server kami dalam waktu 7 hari setelah pesanan selesai dicetak atau dibatalkan.
  • Kami tidak akan membaca, menyalin, mendistribusikan, atau membagikan isi dokumen Anda kepada pihak ketiga manapun.
  • Akses ke file Anda hanya terbatas pada operator cetak yang bertugas memproses pesanan Anda.

4. Keamanan Data

  • Password disimpan dalam bentuk hash terenkripsi dan tidak dapat dibaca oleh siapapun termasuk admin.
  • Pembayaran diproses melalui Midtrans yang tersertifikasi PCI-DSS Level 1, standar keamanan tertinggi untuk pemrosesan pembayaran.
  • Kami tidak menyimpan informasi kartu kredit/debit atau detail rekening bank Anda.
  • Komunikasi antara browser dan server dilindungi oleh enkripsi SSL/TLS.

5. Berbagi Informasi dengan Pihak Ketiga

Kami tidak menjual atau menyewakan data pribadi Anda. Informasi hanya dibagikan kepada:

  • Midtrans: Sebagai payment gateway untuk memproses pembayaran Anda (nama pengguna dan jumlah transaksi).
  • Pihak Berwenang: Jika diwajibkan oleh hukum yang berlaku di Indonesia.

6. Hak Pengguna

  • Anda dapat mengubah informasi akun kapan saja melalui halaman Profil.
  • Anda dapat meminta penghapusan akun dan semua data terkait dengan menghubungi kami.
  • Anda berhak mengetahui data apa saja yang kami simpan tentang Anda.

7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan akan dipublikasikan di halaman ini dengan tanggal pembaruan terbaru.
